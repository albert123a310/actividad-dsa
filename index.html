Tema 1: Completa la siguiente tabla

Concepto Descripción Concepto Descripción
Etiqueta <table> Se utiliza para crear una tabla en HTML. Contiene filas (<tr>) y columnas (<td> o <th>). Etiqueta <form> Se usa para crear formularios, permitiendo a los usuarios ingresar datos.
Atributos de <table> Define el comportamiento y apariencia de la tabla. Algunos atributos comunes son border, cellpadding, cellspacing, width. Etiqueta <input> Permite crear campos donde el usuario puede ingresar datos, como texto, contraseñas, y más.
Etiqueta <tr> Define una fila dentro de una tabla. Etiqueta <label> Se utiliza para definir etiquetas para los elementos de formulario, asociando texto con un campo específico.
Atributo de <tr> Es un atributo que describe características específicas de las filas, como la alineación de su contenido. Etiqueta <select> Crea un menú desplegable para seleccionar una opción entre varias.
Etiqueta <td> y <th> <td> es para definir celdas en las filas, mientras que <th> se usa para las celdas de encabezado. Etiqueta <textarea> Proporciona un área de texto para que el usuario ingrese un texto largo.
Atributos de <td> y <th> Atributos comunes incluyen colspan (para fusionar columnas) y rowspan (para fusionar filas). Etiqueta <fieldset> Agrupa elementos relacionados dentro de un formulario y es útil para crear secciones dentro de un formulario.
Tablas avanzadas Son tablas que incluyen características complejas como filtros, ordenación de datos, paginación, etc., generalmente implementadas con JavaScript. Etiqueta <meter> Se utiliza para representar datos medibles, como un valor dentro de un rango (por ejemplo, para mostrar progreso o intensidad).
Etiqueta <thead> Se utiliza para agrupar el encabezado de la tabla, proporcionando un contexto claro para los datos de la tabla. Etiqueta <progress> Representa el progreso de una tarea, como una barra de progreso.
Etiqueta <tfoot> Agrupa el pie de la tabla, usualmente contiene totales o resúmenes de la información que aparece en la tabla. Etiqueta <datalist> Define un conjunto de opciones predefinidas para un campo <input>, permitiendo autocompletar al usuario.
Etiqueta <tbody> Agrupa el cuerpo de la tabla, separando los datos del encabezado y pie. Etiqueta <summary> Se utiliza junto con la etiqueta <details>, proporcionando un resumen que el usuario puede abrir o cerrar para ver más detalles.

Tema 2: Estructuración de una página web

  • Contenedor (Wrapper): Es el elemento principal que encapsula el contenido de toda la página web, proporcionando una estructura organizada.
  • Cabecera (Header): Contiene el logotipo, título del sitio y enlaces de navegación.
  • Contenido (Content): Es el área donde se muestra la información principal de la página.
  • Menú (Menu): Es una lista de enlaces para navegar por la página.
  • Pie (Footer): Contiene información adicional como derechos de autor y enlaces importantes.
  • Lateral (Sidebar): Barra lateral que contiene menús o contenido secundario.
  • Etiqueta <div>: Contenedor genérico para agrupar elementos sin aplicar estilo por sí mismo.
